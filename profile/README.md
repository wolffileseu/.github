## 🎮 What is Wolffiles?

[**wolffiles.eu**](https://wolffiles.eu) is a community platform for **Wolfenstein: Enemy Territory** (incl. *ET: Legacy*) and **Return to Castle Wolfenstein** — grown out of the German-speaking and wider European ET scene.

We bring together everything that keeps the community alive in one place: a huge file archive, a live server tracker with its own ranking system, player & clan profiles, tutorials, a wiki, and a whole stack of tools that we build and run ourselves.

### What the platform does

- 📦 **File archive** — over 4,900 files and more than 169 GB of maps, mods, configs and tools
- 📡 **Live server tracker** — with ELO/SPM rankings plus player and clan statistics
- 🗺️ **3D BSP map viewer** & **Omni-Bot waypoint browser** right in your browser
- ⚙️ **Config browser** including an ETLegacy converter
- 🚀 **FastDL system** for fast in-game downloads
- 🎥 **ETTV / demo streaming**
- 🖥️ **Game server hosting** via Pterodactyl
- 🏆 **Map of the Week** (automated) plus forum, wiki & tutorials
- 🤖 **Discord & Telegram bots** for server status, search and more

## 🗂️ Our repositories

|Repository                                                                 |Description                                                                            |Stack            |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------|-----------------|
|**[wolffiles-app](https://github.com/wolffileseu/wolffiles-app)**          |The main platform behind wolffiles.eu — archive, tracker, forum, wiki & admin          |`Laravel` · `PHP`|
|**[wolffiles-uploader](https://github.com/wolffileseu/wolffiles-uploader)**|Desktop uploader for fast, deduplicated file uploads                                   |`WinUI 3` · `C#` |

|**[etds](https://github.com/wolffileseu/etds)**                            |Our fork of the ET Dedicated Server (GPL source release) with dual-port & multi-tracker|`C`              |
|**[etlegacy](https://github.com/wolffileseu/etlegacy)**                    |Fork of ET: Legacy, the open-source project based on W:ET (GPLv3, 2010)                |`C`              |
|**[iortcw](https://github.com/wolffileseu/iortcw)**                        |RtCW fork (ioquake3 merge) with server-side weapon-stats tracking                      |`C`              |
|**[omnibot-waypoints](https://github.com/wolffileseu/omnibot-waypoints)**  |Mirror of the Omni-Bot waypoints (Assembla SVN r4032) for Enemy Territory              |`Python`         |
|**[gameserver](https://github.com/wolffileseu/gameserver)**                |Container setup for our game servers                                                   |`Dockerfile`     |


> 💡 The forks serve as a base for our own patches and community builds — e.g. tracking extensions and bugfixes for modern server setups.
